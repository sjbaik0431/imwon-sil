## 🤖 자동 추출 카테고리

업종별로 본문에서 다음 한국어 키워드를 자동 인식:

| 카테고리 | 인식 키워드 |
|---|---|
| **재무** (공통) | 매출, 영업이익, 영업이익률, 현금잔고, 매출채권, 여신한도잔여 |
| **인사** (공통) | 재직자수, 이직률, 연차소진율, 결재 대기 |
| **🏭 제조 — 수주·영업** | 일일 수주액, 월 일평균, 수주 누계, 일본·해외영업 수주, 신규 수주 |
| **🏭 제조 — 생산·품질** | 필요 일수, 가동률, 납기준수율, 수율, 불량률, 재고일수, 품질 이슈·AS 사항 |
| **🏨 임대 — 객실·점유율** | 객실 점유율(OCC), ADR, RevPAR, 임대료 수금, 공실률, 관리비 |
| **🎯 프로젝트** | 응찰, 낙찰, 낙찰률, 수주잔고, 진행 프로젝트, 진행률 |
| **일정** | "5/10 원천세 신고", "2026-05-15 미팅" 같은 날짜+제목 |

## 🔐 데이터 격리

- 각 카피본(`/v01-mfg/`~`/v10-proj/`)은 LocalStorage 키에 `imwon.v01-mfg.*` ~ `imwon.v10-proj.*` prefix를 자동 적용
- 회사 1이 `/v01-mfg/`에 입력한 모든 데이터는 다른 카피본에서 보이지 않음
- 첫 진입 시 환영 모달 1회 표시 후 빈 상태로 시작
